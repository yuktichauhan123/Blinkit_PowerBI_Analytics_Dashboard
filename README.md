#Blinkit Business Intelligence Dashboard | Power BI

Project Type: End-to-End BI Dashboard
Tool Used: Power BI
Domain: Quick Commerce / Retail Analytics

Project Overview
This project is a Blinkit-inspired Business Intelligence dashboard designed to analyze business performance across Sales, Customers, Marketing, Feedback, and Delivery operations.
The focus of this project is to convert raw data into actionable business insights using interactive visuals, slicers, and DAX measures.

Key Objectives
Track Year-to-Date (YTD) performance
Analyze order quantity & value trends
Monitor delivery efficiency and delays
Identify top-performing categories and areas
Enable dynamic analysis using filter panels
