# Blinkit Business Intelligence Dashboard (Power BI)

Turning raw grocery delivery data into **meaningful business insights**.
This project is a **Blinkit-inspired Power BI dashboard** built to analyze operational performance and support better decision-making using data ‚Äî not just visuals.

## What this dashboard does

Tracks **Year-to-Date performance**
Analyzes **sales quantity & trends**
Evaluates **delivery efficiency and delays**
Highlights **top categories, areas, and customers**
Enables quick analysis using interactive filters

## Why this project?

Quick-commerce businesses generate a lot of data, but decisions need **clear answers**, not raw numbers.
This dashboard focuses on:
*What is performing well?*
*Where are delays happening?*
*Which categories and areas need attention?*

## Key Features

Interactive **filter panel** (Date, Category, Area, KPI toggle)
Date analysis: **YTD, Last Month, Last 3/6/9 Months**
Toggle between **Quantity & Value**
Delivery analysis using **Promised vs Actual Time**
Business-focused KPIs and trends
